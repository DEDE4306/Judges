# 最高法院特别法庭会议纪要  
**案件编号**：SP-1950-0042  
**案件名称**：洞穴探险者谋杀案  
**记录时间**：1950年5月18日  
**书记官**：最高法院书记处  

---

## 一、法官立场摘要  

### 1. 首席法官 Truepenny.C.J  
- **立场**：有罪（死刑）  
- **核心观点**：  
  - 严格遵守刑法条文"任何人故意夺取他人生命应处以死刑"  
  - 威特莫尔撤回同意的行为使抽签结果无效  
  - 建议行政赦免但坚持司法独立判决  

### 2. 自然法法官 Foster.J  
- **立场**：无罪  
- **核心观点**：  
  - 洞穴环境构成"自然状态"，原有社会契约失效  
  - 抽签程序体现原始契约精神  
  - 法律的根本目的是保护生命而非机械惩罚  

### 3. 法律实证主义法官 Keen.J  
- **立场**：有罪（死刑）  
- **核心观点**：  
  - 被告行为完全符合故意杀人罪构成要件  
  - "极端生存状态"不是法定免责事由  
  - 法官无权创设法律例外  

### 4. 实用主义法官 Handy.J  
- **立场**：无罪  
- **核心观点**：  
  - 常识判断应优先于法律条文  
  - 抽签程序具有原始正当性  
  - 公众良知不认同机械适用谋杀罪条款  

### 5. 内心冲突法官 Tatting.J  
- **立场**：有罪（建议特赦）  
- **核心观点**：  
  - 承认被告存在法律意义上的杀人故意  
  - 极端情境导致量刑困境  
  - 建议通过特赦程序实现个案正义  

---

## 二、法官间分歧与共识  

### 主要分歧点  
1. **法律解释方法论**：  
   - 文本主义（Truepenny.C.J/Keen.J）vs 自然法（Foster.J）vs 实用主义（Handy.J）  
2. **同意有效性**：  
   - 威特莫尔撤回同意是否使抽签无效（5位法官存在3种不同解读）  
3. **极端情境的法律效力**：  
   - 是否构成"自然状态"（Foster.J）vs 仅是量刑因素（Tatting.J）vs 无关因素（Truepenny.C.J）  

### 有限共识  
1. 案件具有特殊性  
2. 威特莫尔最初提议抽签的事实无争议  
3. 现行刑法条文未明确规定此类极端情形  

---

## 三、关键法律焦点  

1. **犯罪构成要件争议**：  
   - 被告在饥饿状态下是否具备完全刑事责任能力  
   - "代替掷骰子"行为是否构成刑法意义上的"故意"  

2. **同意效力边界**：  
   - 威特莫尔先提议后撤回的法律后果  
   - 多数人意志能否推翻个人撤回决定  

3. **法律漏洞处理**：  
   - 成文法未预见情形下应严格解释还是目的解释  
   - 紧急避险制度是否可类推适用  

4. **司法权边界**：  
   - 法官是否可通过解释创设例外  
   - 司法判决与行政赦免的职能划分  

---

## 四、最终表决结果  

| 法官            | 表决意见       | 附带建议         |  
|-----------------|--------------|------------------|  
| Truepenny.C.J   | 有罪（死刑）   | 提请行政赦免     |  
| Foster.J        | 无罪          | 无               |  
| Keen.J          | 有罪（死刑）   | 无               |  
| Handy.J         | 无罪          | 无               |  
| Tatting.J       | 有罪（死刑）   | 立即启动特赦程序 |  

**裁定结论**：  
3:2 通过有罪判决，依法判处四名被告死刑。根据《最高法院特别程序法》第7条，本案将同步启动以下特别程序：  
1. 向行政长官提交全体法官签名的赦免建议书  
2. 将本案法律争议提请立法机关进行释法  
3. 建立极端情境刑事案例指导档案（编号：EG-1950-001）  

**闭庭时间**：1950年5月18日16:30  
（本记录经所有出席法官确认无误）
